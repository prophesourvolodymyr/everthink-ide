# Everthink IDE

An AI coding assistant written in Rust. Like OpenCode, but with our custom V6 system.

## What Is This?

Everthink IDE is an AI coding tool that combines:
- Chat interface (like OpenCode)
- Autonomous building
- Skills system (Library / Installed / Favorites)
- 22+ LLM providers
- Memory + Progress + Decisions tracking
